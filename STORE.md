# Chrome Web Store Listing

## Title
VIP Video Parser

## Short description (≤132 chars)
One‑click VIP video parsing. Choose a parser and play in a new tab. Simple, fast, configurable.

## Full description
A clean, intuitive Chrome extension for one‑click VIP video parsing.

- Pick a parser in the popup (preloaded examples: Xiaomi, Bingdou)
- Click Parse & Play to open the parser page in a new tab
- Right‑click a page or link to use the default or other parsers
- Add or edit parser prefixes in Options (ideally end with `?url=`)
- The target URL is encoded once automatically

Highlights
- Simple and fast: one primary action
- Named parsers for clarity
- New‑tab playback avoids site CSP/cross‑origin issues
- No broad host permissions for manual triggers

Notes
- Personal study and research only. Ensure compliance with target sites’ Terms and local laws.
- Third‑party parser sites may change or be unavailable; switch or add another parser in Options.

Permissions
- activeTab (read the active tab URL after user action)
- tabs (open a new tab)
- storage (persist settings)
- contextMenus (right‑click entry)
- scripting (fallback to read location.href if needed)
